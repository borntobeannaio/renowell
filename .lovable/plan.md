

## План: Пирамида на полную ширину на мобильных

На мобильных (< 768px) все слои пирамиды будут занимать полную ширину экрана вместо градуированных процентных ширин. На десктопе — без изменений.

### Изменения

**`src/components/modules/brandhub/InteractivePyramid.tsx`**

Сейчас каждый слой имеет фиксированную ширину через `style` или className (`w-[40%]`, `w-[52%]`, ...`w-full`). 

Изменение: использовать `useIsMobile()` хук и на мобильных задавать всем слоям `w-full`, а на десктопе оставить текущие проценты.

```tsx
import { useIsMobile } from "@/hooks/use-mobile";

// внутри компонента:
const isMobile = useIsMobile();

// в className кнопки:
className={`
  ${isMobile ? "w-full" : layer.width}
  ...
`}
```

Один файл, минимальное изменение.
